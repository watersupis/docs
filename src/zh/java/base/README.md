# Java基础知识

本目录系统整理了Java语言的基础知识，适合初学者和有一定经验的开发者查阅。内容涵盖JDK安装、语法基础、IO流、多线程、反射、网络编程等核心主题，配有丰富的示例和配套资源。

---

## 目录导航（推荐学习顺序）

1. [JDK的安装与配置](01_JDK的安装与配置.md)：Java开发环境的安装与环境变量配置方法。
2. [Java基础](02_Java基础.md)：Java语法基础、关键字、命名规范、数据类型等核心知识。
3. [IO流](04_IO流.md)：Java中输入输出流的基本概念、常用类和操作方法。
4. [多线程](05_多线程.md)：多线程的创建方式、线程同步、并发编程基础。
5. [反射](06_反射.md)：Java反射机制、Class对象、动态操作类与方法。
6. [网络编程](07_网络编程.md)：Java网络通信基础，Socket等相关知识。
